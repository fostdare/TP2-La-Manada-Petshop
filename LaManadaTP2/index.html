Presentación del proyecto

Este portafolio reúne los proyectos desarrollados para el Trabajo Práctico N°2 de Práctica Profesionalizante I. La temática elegida fue La Manada Pet Shop, un emprendimiento ficticio orientado a productos y servicios para mascotas.

El objetivo fue aplicar HTML5, CSS y Bootstrap para construir interfaces web estáticas, responsivas y organizadas.

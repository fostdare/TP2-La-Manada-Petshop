# La Manada Pet Shop - TP2 Práctica Profesionalizante I

Proyecto académico desarrollado para la materia Práctica Profesionalizante I de la Tecnicatura Superior en Desarrollo de Software del Instituto Superior Manuel Belgrano.

## Integrantes

- Álvaro Núñez
- Juan Benegas

## Temática

La temática elegida es un pet shop ficticio llamado **La Manada Pet Shop**.  
El objetivo del proyecto es presentar un portafolio digital con diferentes piezas web desarrolladas con HTML5, CSS y Bootstrap.

## Proyectos incluidos

### 1. Portafolio digital

Archivo: `index.html`

Contiene la presentación general del trabajo, integrantes, roles, proyectos desarrollados y bitácora de colaboración.

### 2. Formulario con validación

Archivo: `formulario.html`

Formulario de contacto para clientes de La Manada Pet Shop.  
Incluye campos obligatorios, validación con Bootstrap y simulación de envío.

Tecnologías utilizadas:

- HTML5
- Bootstrap
- JavaScript básico

### 3. Landing page responsive

Archivo: `landing.html`

Landing page promocional de La Manada Pet Shop.  
Incluye menú hamburguesa, carrusel navegable y cuadros modales.

Tecnologías utilizadas:

- HTML5
- Bootstrap
- CSS

### 4. Guía de estilo

Archivo: `guia-estilo.html`  
Archivo CSS: `css/guia-estilo.css`

Guía visual con paleta de colores, tipografía, acordeón de textos, campo de formulario con foco, botones 3D y enlaces animados.

Tecnologías utilizadas:

- HTML5
- CSS
